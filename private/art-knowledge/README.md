このディレクトリは非公開の芸術ナレッジを保存するための領域です。

目的
- 研究ノート、リファレンス、アーカイブ、草稿などをここに蓄積します。
- 直接サイト（HP）には掲載しません。公開用コンテンツは別の場所に移してください。

利用ルール（任意）
- ファイルはカテゴリ別にサブディレクトリを作成して整理してください（例: theory, artists, exhibitions, techniques）。
- 公開が必要になったコンテンツは、確認後に `src/pages` など公開用の場所に移してください。
- 機密情報や第三者の著作物の未許可転載は保存しないでください。

例構成
- private/art-knowledge/
  - README.md
  - theory/
  - artists/
  - exhibitions/
  - references/

必要なら、テンプレートや初期のサブディレクトリを作成します。希望があれば教えてください。
